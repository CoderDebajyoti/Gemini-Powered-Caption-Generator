A powerful and intuitive web application built with **Streamlit** that uses **Google Gemini 1.5 Pro** to generate a complete social media post package—including captions, emojis, hashtags, and song suggestions—from a single image.

---

## ✨ Features

**Gemini Powered Caption Generator** is designed as a one-stop solution for content creators, influencers, and marketers:

* **🤖 AI-Powered Content Generation** – Uses Google’s **Gemini 1.5 Pro** for deep multimodal image understanding.
* **📝 Customizable Captions** – Choose between short, moderate, or long & detailed captions.
* **🎵 Mood-Based Song Suggestions** – Includes clickable YouTube Music search links.
* **🎨 Complete Post Package** – Captions, emojis, and hashtags in one go.
* **📋 One-Click Copy** – Instantly copy generated text to your clipboard.
* **🌐 Modern UI** – Responsive, clean interface built with Streamlit.

## 🚀 Getting Started

### Prerequisites

* **Python 3.9+**
* **Google Gemini API Key** – [Get one here](https://aistudio.google.com/app/apikey)

### Installation

1. **Clone the Repository**

   ```bash
   git clone https://github.com/CoderDebajyoti/CaptionGen.git
   cd CaptionGen
   ```

2. **Create & Activate a Virtual Environment** *(Recommended)*

   ```bash
   # Windows
   python -m venv venv
   venv\Scripts\activate

   # macOS/Linux
   python3 -m venv venv
   source venv/bin/activate
   ```

3. **Install Dependencies**

   ```bash
   pip install -r requirements.txt
   ```

### Running the App

```bash
streamlit run app.py
```

Your browser will automatically open the app’s local address.
